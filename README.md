# Music App

## Built using clean Architecture

- Application:: Context
- Views:: React
- Infrastructure:: (Axios)

## Other Features

- Login resides in Context App State
- Uses the Deezer API from `https://developers.deezer.com/`
- Bootstraped using `create-react-app`

## More About the App

### Libraries

- axios
- node-sass
